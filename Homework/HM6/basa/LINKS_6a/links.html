Способ №1. <a>

Создаём с помощью тега < a > список из пяти элементов, что автоматически приводит их в горизонтальное положение. Далее, с помощью border - right ; добавляем вертикальные полосы и окрашиваем их в красный цвет. Последней черте, закрывающей пятую сылку, задаём border - right: none ; с помощью селектора а: last - child { }

Способ №2. li {display: inline;}

Создаём горизонтальное меню навигационнных ссылок с помощью элемента < ul >, < li >, путём стилизации обычному списку, свойства - display, с присвоением значения - inline, чтобы пункты списка распологались друг за другом. Переопределяем пункты списка с блочных на строчные и добавляем стилевое оформление для нащего горизонтального меню. Далее, с помощью селектора CSS и псевдоэлемента ::after { } добавляем вертикальную черту content: " | " ; и дополнительно стилизуем, придавая нужный нам вид. После всего, убираем последнюю черту, закрывающую пятую ссылку с помощью :last - child ::after { content: " " ;}.

Для добавления разделительной полосы, можно оставить и border - right: 3px solid red; как в первом варианте.
Также, второй вариант для расположения вертикальных полос (разделителей), немного проще, чем вышеописанный. Можно в список < li >, додбавить через свойство content и селектора li + li ::before; он выберет все элементы < li > кроме первого и мы не добавляем ещё один селектор :last - child ::after { content: " " ;},

Способ №3. li {float: left;}

Создаём горизонтальное меню навигационнных ссылок с помощью элемента < ul >, < li >. Делаем элементы списка плавающими, путём стилизации списка с присвоением значения - left: float, в результате они расположатся по горизонтали. Высота блока-контейнера < ul > становится равной нулю. Далее, с помощью селектора .three a { } добавляем вертикальный разделитель, и дополнительно стилизуем, через border-right: 3px solid red;. Далее, последнему элементу строчного списка создаём дополнительный класс .three .last в котором приписываем border-right: 0; и с помощью которого убираем последнюю черту, закрывающую пятую ссылку.

Способ №4. li {display: inline-block;}

Создаём горизонтальное меню навигации с помощью элемента < ul >, < li >, путём стилизации списка с присвоением значения - .four li { display: inline - block; }, делаем элемент списка строчно-блочным. Элементы распологаются по горизонтали, в которых с правой стороны образуется промежуток. Добавляем для ссылок a { display :block } после чего, добавляем вертикальную черту с помощью селектора .four li + li ::before { content: " | " ; } он выберет все элементы < li > кроме первого и далее стилизуем их, придавая нужный нам вид.

При добавлении в селектор для ссылок a { } display :block, все изначально выстроенные у меня в горизонталь элементы - расформировались. Ссылки и вертикальные разделители разхехались в отдельные строки. После чего, открыв среду разработчика было видно, что элемент display :block уже находится в body. После всех проведенных манипуляций, я удалил из селектора a { } display :block и все элементы приобрели правильную форму горизонтального меню навигации.

Способ №5. ul {display: flex;}

Создаём горизонтальное меню навигационнных ссылок с помощью элемента < ul >, < li >. Делаем элементы списка < ul > гибким контейнером с поиощью .five ul { display: flex; } в результате элементы списка распологаются горизонтально. Добовляем для ссылок .five a { display: block; } Далее, с помощью селектора .five a { } добавляем вертикальный разделитель, и дополнительно стилизуем, через border-right: 3px solid red;. Далее, последнему элементу строчного списка создаём дополнительный класс .five .last { } в котором приписываем border-right: 0; с помощью которого убираем последнюю черту, закрывающую пятую ссылку и стилизуем.
